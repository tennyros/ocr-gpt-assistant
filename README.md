# OCR-GPT Assistant

A simple assistant for screen text recognition.

## Requirements

- Python 3.7+
- [Tesseract OCR](https://tesseract-ocr.github.io/) must be installed on your system.

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 main.py
```
This will capture your screen and print the recognized text.
